Rails g model Post title:string content:text  user_id:string

And then in your model

	
class Post < ActiveRecord::Base
	belongs_to: user 
end

The right way of doing this is



Rails g model Post title:string content:text  user:belongs_to

This way the model will be created with the belongs_to association there already!

Remember as devs we want to be fast so we can move on to more awesome code ! Don't make more work for yourself !
